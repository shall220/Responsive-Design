Accessible Navigation Bar

For this we can look at the Navigation Bar constructed for this site. Below are images of the Navigation Bar and how it reacts with different resolutions.

What makes this accessible? If you look at each image you can see that the Navigation Bar changes to suit the user. This allows people with any type of device to access the page and view it comfortably. This is responsive design at work. To further help accessibility the Navigation Bar reacts when the mouse hovers over a button so the user knows when they can select it. Once a button has been selected it stays highlighted to let the user know which page they are currently on.

Accessible Images

Here we can look at the images used for the sites logo and motto above the Navigation Bar.

The logo and motto both scale to the users needs, they stay the same size for most resolutions on a desktop. When they are displayed to a mobile user they are in a larger format so the user can see them properly.

The images have high contrast and are easy to make out in black and white.

They are very small in size making them light on bandthwidth being perfect for mobile viewing where connections are slow or usage is a concern.

When zoomed right in there is no pixilation of the image, this way it can still be read/viewed nicely.

When the images cannot be displayed there is simple text to display in their place.

When viewing on mobile everything is "stacked" on top or under everything else making navigation a whole lot of scrolling, the intra-page navigation links help alleviate that.

Accessible Text

Let's look at the text on this page for our example, I chose it because of its readability.

  • There is no confusion over letters. The capital I, the lower case l and the 1 all have discernible features and are much less likely to be confused with each other.
  • The fonts is also unirversal to help try and prevent clutter or confusion. There is no swapping between fonts or font variations.
  • The font has a high contrast to the background ensuring it can be seen easily.
  • The font size is slightly larger than the defualt value to try and make sure there is no strain trying to read when the page loads naturally.

All these points help with accessibility, by limiting any factors that may may hinder peoples consumer experience.
